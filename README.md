# Ingress Comm Automatically send message

### Tips: Bin

Linux:

```sh
$ chmod +x Ingress & ./Ingress -time=5 -sleep=120
```

Windows:

```bat
.\Ingress.exe -time=5 -sleep=120
```

### Configuration Information:
